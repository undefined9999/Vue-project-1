 
   执行git clone https://github.com/wuxin1210/Vue-project-1.git 

1> 执行 npm/cnpm install 			     安装所有组件
2> 执行 cnpm install vuex --save      	     安装vuex所需要组件
3> 执行 cnpm install axios --save            安装axios组件
4> 执行 cnpm install vue-router --save	     安装router组件
5> 执行 cnpm install mint-ui --save    	     安装mint-ui组件
6> 执行 cnpm install node-sass sass-loader --save    安装sass编译的组件
        cnpm install vuex axios vue-router mint-ui node-sass sass-loader --save
